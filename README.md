## 🌟 **Overview**

**IDLEB X** is a complete e-commerce solution that combines a **modern web store** (React + Tailwind CSS) hosted on **Netlify** with a **powerful Telegram bot** hosted on **Render**. The platform is designed specifically for selling digital products, cybersecurity tools, software licenses, courses, and more.

### 🎯 **Key Features**

| Feature | Description |
|---------|-------------|
| 🤖 **Telegram Bot** | Full-featured shop bot with inline keyboards |
| 🌐 **Web Store** | Modern React store with PWA support |
| 📦 **Digital Products** | Support for links, files, apps, and text |
| 💰 **Wallet System** | User balance management |
| 🏷️ **Coupons** | Discount codes (percentage/fixed) |
| 📤 **Referral System** | Invite friends, earn bonuses |
| 🛒 **Shopping Cart** | Multi-product purchases |
| 📊 **Analytics** | Sales statistics and charts |
| 👑 **Admin Panel** | Complete product/user management |
| 📱 **PWA Ready** | Install as mobile app |
| 🌙 **Dark Mode** | Light/Dark theme toggle |
| 💾 **Backup/Restore** | JSON data export/import |
